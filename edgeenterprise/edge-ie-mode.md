## <span data-ttu-id="7c92c-106">IE モードとは</span><span class="sxs-lookup"><span data-stu-id="7c92c-106">What is IE mode?</span></span>

<span data-ttu-id="7c92c-107">Microsoft Edge で IE モードを利用すると、組織が必要とするすべてのサイトを 1 つのブラウザーで簡単に使用できます。</span><span class="sxs-lookup"><span data-stu-id="7c92c-107">IE mode on Microsoft Edge makes it easy to use all of the sites your organization needs in a single browser.</span></span> <span data-ttu-id="7c92c-108">これは、最新のサイトには、組み込まれている Chromium エンジンを使用し、従来のサイトには、Internet Explorer 11 (IE11) から Trident MSHTML エンジンを使用します。</span><span class="sxs-lookup"><span data-stu-id="7c92c-108">It uses the integrated Chromium engine for modern sites, and it uses the Trident MSHTML engine from Internet Explorer 11 (IE11) for legacy sites.</span></span>

<span data-ttu-id="7c92c-109">サイトが IE モードで読み込まれると、IE ロゴ インジケーターがナビゲーション バーの左側に表示されます。</span><span class="sxs-lookup"><span data-stu-id="7c92c-109">When a site loads in IE mode, the IE logo indicator displays on the left side of navigation bar.</span></span> <span data-ttu-id="7c92c-110">IE ロゴ インジケーターをクリックすると、以下のように詳細情報を表示できます。</span><span class="sxs-lookup"><span data-stu-id="7c92c-110">You can click the IE logo indicator to display additional information, as shown:</span></span>

  ![IE ロゴ インジケーター](./media/ie-mode/ie-logo-indicator1.png)

<span data-ttu-id="7c92c-112">(ポリシー経由で) 個別に構成したサイトのみが IE モードを使用し、他のすべてのサイトは、最新の Web サイトとして表示されます。</span><span class="sxs-lookup"><span data-stu-id="7c92c-112">Only those sites that you specifically configure (via policy) will use IE mode, all other sites will be rendered as modern web sites.</span></span> <span data-ttu-id="7c92c-113">サイトで IE モードを使用するには、次の操作を行う必要があります。</span><span class="sxs-lookup"><span data-stu-id="7c92c-113">For a site to use IE mode, you need to:</span></span>

- <span data-ttu-id="7c92c-114">以下のいずれかのポリシーで定義されているエンタープライズ モード サイト リスト XML で、サイトをリストします。</span><span class="sxs-lookup"><span data-stu-id="7c92c-114">List the site in the Enterprise Mode Site List XML defined in one of these policies:</span></span>
